# Project Summary
This project is a multilingual travel assistant web application that enables seamless communication for travelers by providing real-time translation of common phrases. It utilizes AI-powered Optical Character Recognition (OCR) to capture images, extract text, and translate it between Chinese and multiple languages including English, Italian, Spanish, Korean, Japanese, and Russian. Key features include auto-translation, saving favorite phrases, and an intuitive user interface, making it an indispensable tool for overcoming language barriers during travel.

# Project Module Description
- **Backend**: Responsible for managing API requests, data storage, and integration of AI services for translation and OCR. It features enhanced error handling and performance optimizations for reliability. The translation capabilities now support translating from any language to Chinese and return text with bounding box positions for improved visual context.
- **Frontend**: Provides an interactive user interface for entering phrases, viewing translations, managing favorites, and utilizing the camera translation feature. Recent upgrades include improved image upload handling, automatic language detection, and enhanced functionality for overlaying translations on captured images.

# File Description Inventory
- `backend/main.py`: Entry point for the backend service.
- `backend/routers/ocr.py`: API endpoint for OCR services, including functionality for extracting text with positions.
- `backend/services/ocr.py`: Logic for extracting text from images using AI, now includes functionality for returning bounding box positions of detected text.
- `backend/routers/translate.py`: API endpoint for translation services.
- `backend/services/translate.py`: Logic for translating text using AI, including error handling.
- `frontend/src/pages/FavoritesPage.tsx`: Implements the favorites page with improved language selection for translation.
- `frontend/src/pages/MapPage.tsx`: Displays a map interface for locating nearby facilities, with error handling for Overpass API requests.
- `frontend/src/pages/CameraTranslatePage.tsx`: Implements the camera translation feature, now enhanced to display translations directly on captured images, with improved image size handling and automatic language detection.
- `frontend/src/components/PhraseCard.tsx`: Displays individual phrases with playback functionality.
- `frontend/src/components/BottomNav.tsx`: Navigation bar including the camera translation tab, with updated tab order.
- `frontend/src/pages/Index.tsx`: Main entry point for the frontend application, managing tabs including the camera feature.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Vite
- **AI Integration**: Atoms Cloud for translation capabilities, custom AI for OCR functionality

# Usage
1. **Install Dependencies**: 
   - For the backend: Run `pip install -r requirements.txt`
   - For the frontend: Run `pnpm install`
2. **Build the Project**: 
   - Navigate to the backend directory and build the Python files.
   - For the frontend, run `pnpm run build`.
3. **Run the Application**: 
   - Start the backend service.
   - Launch the frontend application.
